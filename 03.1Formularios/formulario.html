todo los formularios que se manejan en la web utilizando metodos para el envio o respuesta de la informacion, los mas comunes son:


get:este metodo s eencarga de enciar la informacion por la url a saloda estandar del navegador, esto propicia que se pueda cisualizar toda la informacion del formularios
POST este metodo se encarga de enciar la informacion por la url pero de forma oculta, lo que ofrece cierto nivel de seguridad, este metodo se usa comunmente para insertar un nuevo registro dentro del sistema
